## Technologies Used

- HTML5
- Tailwind CSS
- GSAP (GreenSock Animation Platform)
- ScrollTrigger Plugin
- Intersection Observer API

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. No build process required - uses CDN for dependencies
